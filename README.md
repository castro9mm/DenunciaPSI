# DenunciaPSI
Página para recebimento de denúncias anônimas de terceiros e escalonamentoc com análise profissional.

Resumo:
Essa pagina devera disponibilizar um espaco para que um trabalhador vitima de algum tipo de abuso ou infração de lei trabalhista possa expor a empresa e escalar maiores providencias.

Ex:
Um colaborador podera enviar anonimamente suas denuncias trabalhistas para que seja oficializada uma análise profunda dos acontecimentos e alinhamento junto aos orgaos competentes.
